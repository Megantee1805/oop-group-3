FoodHub.SG
======

A health app that recommends healthy food near you!

Run
---

::

    $ export FLASK_APP=foodhubsg
    $ export FLASK_ENV=development
    $ flask init-db
    $ flask run

Or on Windows cmd::

    > set FLASK_APP=foodhubsg
    > set FLASK_ENV=development
    > flask init-db
    > flask run

Open http://127.0.0.1:5000 in a browser.
